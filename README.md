# The Hunt 利用ガイド

このサイトはユーザーの皆さまによる情報の共有で成り立っています。
正確な情報共有と円滑な利用のため、以下の点をご確認ください。

---

## 📊 表示とフィルタリングの仕様

- **タブ管理と絞り込み**
  - **ランクタブ (ALL/S/A/F)**: 指定したランクのモブのみを表示します。
  - **エリアタブ (黄金/暁月/漆黒...)**: 拡張パッケージごとに表示を切り替えられます。
- **ソート（表示順）の優先順位**
  リストは以下の優先順位で自動的に整列され、重要な情報が常に上位に表示されます。
  1. **稼働状況**: メンテナンス等で停止していないモブを優先。
  2. **長時間の未出現 (MaxOver)**: 最長湧き時間を過ぎているものを最優先。
  3. **特殊条件の一致**: 出現条件（月齢・天候等）が現在一致しているSランク。
  4. **出現期待度**: 湧き時間内の経過率（進行度％）が高い順。
  5. **時間順**: 次の最短出現予定時刻が早い順。

## 📈 進行度バーと視覚効果の意味

カードやバーの状態は、現在の湧き状況を正確に表しています。

- **進行度バーの色と挙動**
  - **青（グラデーション）**: 通常の進捗状態です。
  - **濃い青**: 最長湧き時間を経過した状態（MaxOver）です。
  - **赤色の発光（点滅）**: 進行度が **90%** を超え、まもなく最長湧き時間に達することを示します。
- **視覚的な演出**
  - **カード外枠の発光**: 枠がランク色のネオンで光っているカードは、現在「湧き時間（窓）」に入っていることを示します。
  - **白い枠の点滅（Sランク）**: そのモブの出現条件（天候・月齢等）が、**現在ちょうど一致している**ことを示します。
  一致している間は即湧きの可能性があります。

## 🕒 報告に関するルール

- **報告の形式**
  - **S / Fランク**: 誤報告防止のため、詳細入力またはスワイプ操作による報告が必要です。可能な限り正確な討伐時刻を入力してください。
  - **Aランク**: サイドバーのボタンをクリックした瞬間に、現在時刻で即座に報告されます。
- **入力の制限**
  - 未来の時刻は報告できません。
  - 最短時刻より大幅に早い報告はエラーになりますが、更新が必要な場合は「強制送信」にチェックを入れることで送信可能です。

## 📍 湧き位置マップと操作

- **情報のリアルタイム共有**
  マップのチェック状況は、利用している全ユーザー間でリアルタイムに同期されます。
- **モバイル端末での操作**
  移動中の誤操作を防ぐため、スマートフォンやタブレットでは **「2回素早くタップ（ダブルタップ）」** することでチェックを切り替える仕様です。

## 💡 サイトの共通仕様

- **メンテナンス対応**: メンテナンス期間は報告機能が自動的に制限されます。手動での設定は不要です。
- **自動同期**: 全てのデータは自動更新されます。画面をリロードしなくても常に最新の状態が保たれます。
- **ブラウザ設定**: JavaScriptを有効にしていただくことで、すべての機能をご利用いただけます。
