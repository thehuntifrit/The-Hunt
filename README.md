# The Hunt 利用ガイド

このサイトは、ユーザーの皆さまによる情報の共有で成り立っています。
正確な情報の共有と円滑な利用のため、以下のガイドラインをご確認ください。

---

## 📊 表示とフィルタリングの仕様

### タブ管理と絞り込み

- **ランクタブ (ALL/S/A/F)**: ランク毎のモブ一覧に切り替えます。
- **エリアタブ (黄金/暁月/漆黒...)**: それぞれのランクタブを2回押すと開きます。拡張ごとで選択でき、選択状態はブラウザに保存されます。

### ソート（表示順）の優先順位

表示順は以下の優先度に従って自動的に並び替えられます。

1. **時間経過**: 「最大時間超え」 > 「最短REPOP超え（経過%で降順）」 > 「最短REPOP以前（残り時間で昇順）」
2. **時限モブ**: 出現時間内の場合には、上記の条件1ごとの最上位に表示されます。
3. **詳細条件**: 上記の条件内で同一順の場合には 「ランク【 S > A > F 】」 > 「インスタンス番号」 の順で整列されます。

## 📈 進行度バーと視覚効果の意味

モブカードやバーの表示は、現在の湧き状況を視覚的に示しています。

### 進行度バーの色と挙動

- **青色のグラデーション**: 最短REPOP時間を経過し、湧く可能性がある状態です。
- **赤色の発光（点滅）**: 進行度が **90%** を超え、まもなく最長時間に達することを示します。
- **濃い青色**: 最長湧き時間を経過した状態です。

### カード外枠の視覚的演出

- **白色の太枠（Sランク）**: 時間条件（ET・天候・月齢）のあるモブが、今現在、時間を満たしている状態を示しています。

## 🕒 報告に関するルール

### 討伐報告の形式

- **Sランク/FATE**: 「報告」ボタンから入力フォームが開きます。可能な限り正確な討伐時刻を入力してください。
- **Aランク**: 「報告」ボタンを押すと即座に現在の時刻で報告されます。

### 入力の制限

- 未来の時刻は報告できません。
- 最短時刻より早すぎる報告はエラーとなります。
- 誤報告の修正などで最短前の時間報告をする場合には、「修正する」にチェックを入れることで送信が可能になります。

## 📍 湧き位置マップと操作

### 情報のリアルタイム共有

マップのチェック状況は、利用しているすべてのユーザー間でリアルタイムに同期されます。

### モバイル端末での操作

スマートフォンやタブレットでの誤操作を防ぐため、**1秒以内に2回素早くタップ**することでチェックの切り替えが行えます。

## 💡 サイトの共通仕様

- **自動同期**: すべてのデータは自動更新されます。サイトのリロードの必要はありません。
- **メンテナンス対応**: FF14のメンテナンス期間中は、報告機能が自動的に制限されます。
- **ブラウザ設定**: JavaScriptを有効にしていただくことで、すべての機能をご利用いただけます。
