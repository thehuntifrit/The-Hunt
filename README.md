# The Hunt 利用ガイド

</div>

このサイトは、ユーザー同士の情報共有で成り立っています。
初めての方でも直感的に使いこなせるよう、画面の見方やルールをまとめました。

---

## 🔰 はじめる前に：キャラクター連携

情報の信頼性を保つため、情報の投稿には「Lodestone認証」が必要です。

1. **コードの取得**: サイト内の案内から、あなた専用の「検証用コード（8文字）」をコピーします。
2. **Lodestoneへ貼り付け**: ご自身のLodestoneプロフィール「自己紹介」欄にコードを貼り付けて保存します。
3. **登録完了**: サイトに戻り、ご自身のキャラクターページのURLを入力して「登録」すれば準備完了です。

## 📈 画面の見方：色とサインの意味

各モブのカードに表示されているバーや枠は、現在の「湧き状況」をリアルタイムに示しています。

### 進行度バー（時間の目安）

- **水色〜青**: まだ準備中、もしくはこれから湧く可能性がある時間帯です。
- **青色**: 湧く可能性が非常に高い、いわゆる「チャンスタイム」です。
- **白く光る**: **特殊な湧き条件（天候や時間）が整っている**「激アツ」なサインです。

### 特別なサイン

- **Sランクの白い太枠**: 現在、そのモブを出現させるための条件（天候・時間・月齢など）がすべて揃っている「POP可能」な状態です。

## 📊 リストの並び：賢い自動ソート

リストの順番はシステムが自動で並び替えます。

- **今すぐ確認すべきものが一番上**: 「すでに湧く時間を過ぎているもの」や「今ちょうど条件が整っているもの」が、自動的にリストのトップに並びます。
- **まだ先のものは下の方へ**: 倒した直後のモブや、メンテナンス中で今は湧かないものは、自然とリストの下へと移動します。

## 🔔 予告と通知：チャンスを逃さない

「そろそろ湧きそう」というタイミングを、デスクトップ通知でお知らせします。

- **ヘッダーの「🔜」マーク**: 特殊な条件（天候など）が必要なモブの条件が整う前に画面上部でお知らせします。現地へ向かう目安にしてください。
- **ベルの通知設定**: ヘッダーのベルアイコンをONにすると、湧き条件の開始に合わせて**ブラウザ通知**でお知らせします。
- **通知のタイミング**: 条件が整う「2分前」と「開始時」に届きます。

## 🕒 報告のルール：みんなで最新に

情報の鮮度が命です。倒した後はスピーディな報告をお願いします。

- **Aランク**: 倒した瞬間に「報告」ボタンを押すだけで完了です。
- **Sランク・Fランク**: 「報告」ボタンから開くフォームに、正確な討伐時刻を入力して送信してください。

## 📍 仲間との連携：リアルタイム共有

- **共有メモ**: カード内のメモ欄に書き込んだ内容は、他のユーザーに即座に伝わります。湧かせの進行状況などの共有にご活用ください。
- **マップのチェック**: 湧き場所のチェック状況もリアルタイムで同期されます。
  - **PC**: 右クリックで拡大レンズ。
  - **スマホ**: 1秒以内に2回素早くタップでチェック切り替え。
