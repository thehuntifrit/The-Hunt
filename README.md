# The Hunt 利用ガイド  

このサイトはユーザーの皆さまによる情報の共有で成り立っています。  
正確な情報共有と円滑な利用のため、以下の点をご確認ください。  

---

## 📊 表示とフィルタリングの仕様  

- **タブ管理と絞り込み**  
  - **ランクタブ (ALL/S/A/F)**: 指定したランクのモブを表示します。  
  - **エリアタブ (黄金/暁月/漆黒...)**: 拡張パッケージごとに選択でき、その状態を記憶します。  

- **ソート（表示順）の優先順位**  
  - 最大時間超え > 最短REPOP超え (経過%順) > 最短REPOP以前 (残り時間順)
  - 上記のそれぞれの項目内で、「特殊条件の時間内 > ランク (S>A>F) > インスタンス」で並び替えられる。

## 📈 進行度バーと視覚効果の意味  

モブカードやバーは現在の湧き状況を示しています。  

- **進行度バーの色と挙動**  
  - **青（グラデーション）**: 最短REPOP時間を超えた状態。  
  - **濃い青**: 最長湧き時間を経過した状態。  
  - **赤色の発光（点滅）**: 進行度が **90%** を超え、まもなく最長湧き時間に達する状態。  

- **カード外枠の視覚的演出**  
  - 枠がランク色で光っているカードは、湧き時間に入っていることを示します。  
  - 白枠（Sランク）はそのモブの出現条件（天候・月齢など）が、現在条件を満たしていることを示します。  

## 🕒 報告に関するルール  

- **討伐報告の形式**  
  - **S / Fランク**: モブカードの右側の「報告」ボタンを押すと別窓が開きます。可能な限り正確な討伐時刻を入力してください。  
  - **Aランク**: モブカードの右側の「報告」ボタンを押すと即座に報告されます。  

- **入力の制限**  
  - 未来の時刻は報告できません。  
  - 最短時刻より大幅に早い報告はエラーになりますが、間違った報告などで修正が必要な場合は「修正する」にチェックを入れることで送信可能です。  

## 📍 湧き位置マップと操作  

- **情報のリアルタイム共有**  
  マップのチェック状況は、利用している全ユーザー間でリアルタイムに同期されます。  

- **モバイル端末での操作**  
  スクロール中などの誤操作を防ぐため、スマートフォンやタブレットでは **1秒以内に2回素早くタップ** することでチェックを切り替えることができます。  

## 💡 サイトの共通仕様  

- **自動同期**: 全てのデータは自動更新されます。画面をリロードしなくても常に最新の状態が保たれます。  
- **メンテナンス対応**: メンテナンス期間中の場合は、報告機能が自動的に制限されます。  
- **ブラウザ設定**: JavaScriptを有効にしていただくことで、すべての機能をご利用いただけます。
