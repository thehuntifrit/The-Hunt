# The Hunt 利用ガイド

このサイトは、ユーザーの皆さまによる情報の共有で成り立っています。
正確な情報の共有と円滑な利用のため、以下のガイドラインをご確認ください。

---

## 📊 表示とフィルタリングの仕様

### タブ管理と絞り込み

- **ランクタブ (ALL/S/A/F)**: 指定したランクのモブのみを表示します。
- **エリアタブ (黄金/暁月/漆黒...)**: 拡張パッケージごとに表示を切り替えられます。選択状態はブラウザに保存されます。

### ソート（表示順）の優先順位

表示順は以下の優先度に従って自動的に並び替えられます。

1. **時間経過**: `最大時間超え` > `最短REPOP超え（経過%順）` > `最短REPOP以前（残り時間順）`
2. **詳細条件**: 上記の各区分内で、`特殊条件の時間内` > `ランク（S > A > F）` > `インスタンス順` で整列されます。

## 📈 進行度バーと視覚効果の意味

モブカードやバーの表示は、現在の湧き状況を視覚的に示しています。

### 進行度バーの色と挙動

- **青色（グラデーション）**: 最短REPOP時間を経過し、湧く可能性がある状態です。
- **濃い青色**: 最長湧き時間を経過した状態です。
- **赤色の発光（点滅）**: 進行度が **90%** を超え、まもなく最長湧き時間に達することを示します。

### カード外枠の視覚的演出

- **ランク色の発光**: 湧き時間（最短REPOP経過後）に入っていることを示します。
- **白色の太枠（Sランク）**: そのモブの出現条件（天候・月齢など）を、現在満たしていることを示します。

## 🕒 報告に関するルール

### 討伐報告の形式

- **S / Fランク**: 「報告」ボタンから入力フォームが開きます。可能な限り正確な討伐時刻を入力してください。
- **Aランク**: 「報告」ボタンを押すと即座に現在の時刻で報告されます。

### 入力の制限

- 未来の時刻は報告できません。
- 最短時刻より大幅に早い報告はエラーとなります。
- 誤報告の修正などで特別な入力が必要な場合は、「修正する」にチェックを入れることで送信が可能になります。

## 📍 湧き位置マップと操作

### 情報のリアルタイム共有

マップのチェック状況は、利用しているすべてのユーザー間でリアルタイムに同期されます。

### モバイル端末での操作

スマートフォンやタブレットでの誤操作を防ぐため、**1秒以内に2回素早くタップ**することでチェックの切り替えが行えます。

## 💡 サイトの共通仕様

- **自動同期**: すべてのデータは自動更新されます。リロードの必要はありません。
- **メンテナンス対応**: メンテナンス期間中は、報告機能が自動的に制限されます。
- **ブラウザ設定**: JavaScriptを有効にしていただくことで、すべての機能をご利用いただけます。
