# The Hunt 利用ガイド

</div>
<!-- markdownlint-enable MD033 -->

本サイトは、エオルゼアにおけるモブハント情報の共有および出現時間の管理を行うためのツールです。
ユーザー間でのリアルタイムな情報同期により、効率的な探索を支援することを目的としています。

---

## 🔰 キャラクター連携の必要性

本サイトでは、情報の信頼性を担保し、不適切な投稿を抑制するために、**討伐報告およびメモの投稿に「Lodestone認証」を必須としています。**

- **情報の信頼性**: 報告者の識別を可能にすることで、データの正確性を維持します。
- **コミュニティの保護**: 悪意のあるユーザーによるデータの改ざんや荒らし行為を防止します。

連携手順については、登録ボタンから表示される「ユーザー認証モーダル」の案内に従ってください。

---

## 🧭 アプリナビ：主要機能と通知

画面のアイコンより、各パネルの表示・操作が可能です。

- **📊 RANK**: モブのランク（S/A/F）や拡張パッケージ別のフィルタリングを行います。
- **📢 告知**: 管理者からのお知らせを表示します。
- **🛠️ メンテ**: サーバーメンテナンス予定、およびメンテ明けの出現状況に関する情報を表示します。
- **⚠️ エラー**: システムエラーの詳細を確認できます。
- **🔔 通知**: 音声通知やブラウザ通知の有効・無効を個別に切り替えます。
  - **タイミング**: 出現条件合致の「2分前」および「開始直後」。
  - **環境**: PC（デスクトップ通知）、スマホ（ブラウザ起動中の音声再生）。

---

## 🕒 討伐報告のルールと仕様

REPORT(+)から行える報告機能の仕様です。

- **報告の方法**:
  - **Aランク**: 「REPORT(+)」ボタンからの即時報告（現在時刻）が可能です。
  - **S/Fランク**: 報告フォームに正確な討伐時刻を入力してください。
- **報告の制限と修正**:
  - **時間の整合性**: 最短出現時刻より5分以上早い時刻、または現在時刻より10分以上先の時刻は報告できません。
  - **修正**: 誤った報告を上書きする場合は、正しい時刻を入力し、フォーム内の「修正する」にチェックを入れて送信してください。

---

## 📊 モブリスト：各表記と進行度

各項目には、現在の出現状況を示す各種情報が表示されます。

- **右端の数字（%）**:
  - 前回の討伐（またはメンテ明け）から、次に出現する可能性が高まるまでの経過時間をパーセンテージで表示します。
- **REPORTボタン (+)**:
  - 認証済みユーザーのみ操作可能です。Aランクの即時報告、またはS/Fランクの報告フォーム呼び出しを行います。
- **背面のゲージ（色の遷移）**:
  - 最短時刻（min）を経過するとバーが伸び始め、最大時刻（max）に向かってバーが伸びていきます。
- **白光エフェクト (Condition Active)**:
  - リスト項目が白く点滅している場合、特殊な出現条件（天候・時間・月齢など）がすべて合致しており、**現在出現可能**な状態であることを示します。

---

## 💳 モブカード：情報の補足と座標埋め

モブリストを選択した際に表示される詳細パネル（モブカード）の機能です。

- **共有メモ (memo)**:
  - モブごとに最大30文字の情報を投稿でき、全ユーザーに即時同期されます。
- **出現候補地の調査（座標埋め）**:
  - リスキーモブ（S級等）の出現候補地を調査した結果を記録・共有できます。
  - **モブリストの📍表示**: 未確認の候補地が「📍@3」のようにカウントされ、残り1箇所に特定されると「📍14番」のように地点番号が表示されます。
  - **操作**: PCは右クリックで拡大鏡が表示されます。スマホは1秒以内の2連続タップで、チェックの切り替えが可能です。
